# Smart Hospital Management System

A Java-based academic project developed using Apache NetBeans for learning software engineering and object-oriented programming concepts.

## Features

* Patient record management
* Doctor management
* Appointment handling
* Basic hospital workflow simulation
* File handling concepts

## Technologies Used

* Java
* Apache NetBeans
* OOP Concepts
* File Handling

## Purpose

This project was developed as part of academic learning and practice in software development.

## Project Status

Academic / Learning Project
